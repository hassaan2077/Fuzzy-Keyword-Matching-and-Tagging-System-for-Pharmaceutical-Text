# Intelligent Publication Sentiment Tagging System

Engineered an end-to-end automated classification pipeline to analyze and tag pharmaceutical publications with relevant sentiment topics and keywords.

---

## Key Achievements

* Processed 10,000+ publications with 80%+ accuracy using multi-criteria fuzzy matching algorithms
* Reduced manual content review time by 70% through intelligent automation
* Implemented 4 fuzzy matching algorithms (Partial Ratio, Token Sort, Token Set, WRatio) with adaptive thresholds

---

## Technical Implementation

* Built in Python using pandas, RapidFuzz, and regex for robust text processing
* Designed batch processing system with memory optimization for large datasets (1000+ records per batch)
* Created configurable AND/OR logic matching to handle complex synonym queries
* Developed debug mode with detailed scoring analysis to optimize matching accuracy

---

## Features

* Automated keyword extraction from structured reference data (sentiment topics & synonyms)
* Enhanced text preprocessing for medical/pharmaceutical terminology
* Near-miss analysis to identify and resolve false negatives
* Comprehensive reporting with match statistics and debugging insights
* Backup and error handling for safe data processing
